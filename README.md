# Pearl's Travel Getaway Hub

## 🌍 Description
Pearl's Travel Getaway Hub is a modern, interactive booking system for exploring beautiful tourist destinations in Mindanao, Philippines. Users can browse featured locations, view image galleries, check real-time pricing, and complete bookings with a seamless user experience.

## ✨ Features
- **6 Featured Destinations** - Carefully selected Mindanao tourist spots
- **Image Gallery** - Beautiful photo galleries for each destination with navigation controls
- **Real-Time Price Calculator** - Dynamic pricing based on number of guests and duration
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Smooth Booking Experience** - User-friendly form with automatic price updates
- **Floating Booking Button** - Quick access to booking form from anywhere on the page
- **Booking Confirmation** - Instant booking summary display after submission

## 🚀 How to Use

### Getting Started
1. Open `booking-system.html` in any web browser
2. No installation required - runs locally

### Making a Booking
1. **Browse Destinations** - View all featured tourist spots with descriptions and prices
2. **View Gallery** - Click "Book Now" on any spot to see image gallery
3. **Select Destination** - Choose from the dropdown menu or click a spot card
4. **Fill Details** - Enter your name, email, phone, travel date, number of guests, and duration
5. **Check Pricing** - See automatic price calculation (guests × days × rate)
6. **Confirm Booking** - Click "Confirm Booking" to finalize
7. **Confirmation** - View your booking summary confirmation

### Quick Access
- Use the **floating booking button** (📅 Book Now) at the bottom-right to quickly jump to the booking form from anywhere on the page
